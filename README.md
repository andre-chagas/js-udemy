Repositorio de estudos baseado no curso **"JavaScript e TypeScript do basico ao avancado (JS e TS)"**, com exercicios, exemplos e pequenos projetos desenvolvidos ao longo das aulas.

O foco deste material e consolidar fundamentos de **JavaScript moderno**, evoluir para **TypeScript**, praticar **front-end e back-end** e construir base solida para desenvolvimento **full stack** com ferramentas e bibliotecas amplamente usadas no mercado.

## Visao Geral

Este repositorio acompanha a progressao das aulas e praticas do curso, cobrindo desde os conceitos iniciais de programacao com JavaScript puro ate temas avancados como:

- JavaScript moderno (ES6+)
- TypeScript do basico ao avancado
- Logica de programacao
- Programacao orientada a objetos
- Programacao funcional
- JavaScript assincrono com promises, fetch, ajax e axios
- Node.js, Express e sistema de modulos
- MongoDB com Mongoose
- APIs REST com JWT e Sequelize
- React, React Hooks, Redux e Redux Saga
- Webpack e Babel
- Testes automatizados com Jest
- Expressoes regulares (Regex)
- SOLID e Design Patterns
- Next.js, Strapi, SSR, SSG e ISR
- Bancos de dados SQL com MySQL/MariaDB e Knex
- Conceitos de deploy, Linux, NGINX, PM2, SSL/TLS e GitHub

## Objetivos Do Repositorio

- Registrar a evolucao das aulas e exercicios praticos
- Centralizar exemplos de JavaScript no navegador e no Node.js
- Servir como material de consulta para revisao dos principais conceitos
- Reunir pequenos experimentos, desafios e implementacoes criadas durante o curso

## Conteudos Abordados

### Fundamentos

- Instalacao de ferramentas como Node.js e VS Code
- Variaveis, operadores, estruturas condicionais e de repeticao
- Funcoes, arrays, objetos e manipulacao do DOM
- Logica de programacao e resolucao de problemas

### JavaScript Em Profundidade

- JavaScript orientado a objetos
- Factory functions, funcoes construtoras e classes
- Closures, escopo, hoisting e comportamento da linguagem
- Programacao funcional aplicada com arrays, objetos e funcoes
- Assincronismo com promises, fetch API, ajax e axios

### Front-End

- HTML5 e CSS3 como base complementar
- JavaScript no navegador com projetos praticos
- Webpack e Babel para compatibilidade com navegadores antigos
- React JS basico e avancado
- React Hooks, Redux, Redux Saga e persistencia de estado

### Back-End

- Node.js e Express
- Sistema de modulos do Node.js
- MongoDB e Mongoose
- API REST com MariaDB/MySQL, Sequelize e JWT
- Middlewares, validacoes, organizacao e seguranca da aplicacao

### Arquitetura, Qualidade E Boas Praticas

- TypeScript aplicado ao desenvolvimento moderno
- Principios de POO e SOLID
- Design Patterns (GoF)
- EditorConfig, ESLint e Prettier
- Testes unitarios e de integracao com Jest

### Infraestrutura E Deploy

- Linux e administracao basica de servidor
- NGINX como proxy reverso
- PM2 para gerenciamento de processos Node
- HTTPS com certificado SSL/TLS via Let's Encrypt
- Git, GitHub, chaves SSH e fluxo de publicacao
- Deploy de aplicacoes com foco em seguranca

## Estrutura Atual

O repositorio esta organizado principalmente por pastas de aulas, como:

- `aula10`, `aula11`, `aula12`...
- `aula20`, `aula30`, `aula40` com exemplos de HTML, CSS e JavaScript
- `modelo/` com uma base reutilizavel para exercicios
- `index.js` na raiz com exemplos iniciais

Cada pasta pode conter:

- arquivos `.js` com exercicios e exemplos isolados
- paginas `.html` para execucao no navegador
- arquivos de estilo `.css`
- pequenos scripts para praticar conceitos especificos

## Como Executar

Como este repositorio reune exercicios de diferentes etapas do curso, a execucao varia conforme a aula:

### Executar arquivos JavaScript com Node.js

```bash
node nome-do-arquivo.js
```

Exemplo:

```bash
node aula10/index.js
```

### Executar exemplos no navegador

Abra o arquivo `index.html` da aula desejada diretamente no navegador ou utilize uma extensao como Live Server no VS Code.
